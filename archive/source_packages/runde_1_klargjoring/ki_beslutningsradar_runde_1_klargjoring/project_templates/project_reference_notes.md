# Prosjektreferanser

Relevante regler fra prosjektoppsettet:

- Les `AGENTS.md` og `AI_RULES.md` først.
- Start med minste nyttige artefakt.
- Bevar promptpakker, outputs og reviewspor som filer.
- Nye use cases bør starte som workspace under `workspaces/`.
- Når materialet kommer fra idé, Notion, GitHub eller lokale filer, bruk `concept_context` først.
- ChatGPT Project holder intensjon, research og beslutninger.
- Codex gjør repoarbeid, filer, kode, dokumentasjon og validering.

Anvendt på dette prosjektet:

- Ikke start med app.
- Ikke start med nytt repo.
- Start med `workspaces/ki_beslutningsradar/`.
- Bruk denne zip-pakken som seed for workspace.
