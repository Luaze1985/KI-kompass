# Sluttkontroll før zip

## 1. Dette er klart

- klargjøringsstruktur
- manifest
- README
- agentprompter
- foreløpig kildeoversikt
- mangelliste
- Codex/GitHub-handoff
- prosjektkontekstpakke
- use case-brief

## 2. Dette mangler

- faktiske datafangster fra researchtrådene
- syntese/kritisk review
- full kildeoversikt fra datafangster

## 3. Dette bør ikke forsinke zip

- appdesign
- MVP-stack
- database/skåringsmotor
- komplett GitHub-workspace

## 4. Dette bør forskes videre på senere

- Algorithmic Impact Assessment
- offentlig rettssikkerhet
- kunde/tillit/opplevd verdi
- linse-integrasjon
- decision log schema

## 5. Anbefaling

**Klar for klargjøringszip.**

Ikke klar som ferdig prosjektgrunnlag før datafangstene er produsert.
