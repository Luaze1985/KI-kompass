# Sluttkontroll før zip

## 1. Dette er klart

- Ledertråd og hovedspørsmål.
- Datafangst for målklarhet.
- Supplerende datafangster for separabilitet, forklarbarhet/human oversight og anti-overreliance.
- Foreløpig syntese.
- Foreløpig scoremodell med vektingsalternativer.
- Kildeoversikt.
- Mangelliste og Codex/GitHub-handoff.

## 2. Dette mangler

- Test mot faktiske caser.
- Kalibrering av skåring.
- Egen datafangst om beslutningslogg.
- Operasjonalisering av linser.

## 3. Dette bør ikke forsinke zip

- Endelig appflyt.
- Visuelt design.
- Full repoimplementering.

## 4. Dette bør forskes videre på senere

- Automation bias og appropriate reliance.
- EDPB/Datatilsynet om automatiserte avgjørelser.
- ISO/IEC 23894 og AI-risikostyring.
- Beslutningslogg og auditability.

## 5. Anbefaling

**Klar for zip** som research- og kontekstpakke for runde 1.
