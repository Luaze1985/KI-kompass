# Prompt 02: Separabilitet

Du arbeider med prosjektet KI-beslutningsradar.

Hovedspørsmål: Bør denne delen av beslutningen overlates til KI, støttes av KI, delautomatiseres med menneskelig kontroll, eller forbli menneskelig?

Oppgave: Lag kort, operativ og kildekontrollert materiale for temaet **Separabilitet**. Ikke bygg app. Ikke lag endelig modell. Skill mellom kriterier, kontrollspørsmål, stoppregler, testcaser og kilder.

Leveranse:

1. Kort sammendrag
2. Operativ definisjon
3. Viktigste kilder
4. Kriterier
5. Skåringsspørsmål
6. Røde flagg
7. Eksempler
8. Kildekontroll
9. Handoff til syntese
