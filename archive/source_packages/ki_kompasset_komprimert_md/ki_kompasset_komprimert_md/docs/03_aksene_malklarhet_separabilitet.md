# Aksene: målklarhet og separabilitet

KI-kompasset har to akser.

```text
MÅLKLARHET
I hvilken grad kan målet formuleres som en regel på tvers av situasjoner?

SEPARABILITET
I hvilken grad kan prediksjon brukes uten menneskelig skjønn?
```

## Målklarhet

Målklarhet handler om hvor tydelig og stabilt målet er.

### Høy målklarhet

Målet kan uttrykkes som en regel, terskel eller stabil prioritering.

Eksempler:

```text
minimere arbeidsulykker
finne avvik i MVA-melding
oppdage manglende dokumentasjon
varsle overskredet frist
```

### Lav målklarhet

Målet er sammensatt, kontekstavhengig eller inneholder konflikt mellom hensyn.

Eksempler:

```text
balansere effektivitet og tillit
prioritere rettferdig mellom avdelinger
vurdere om et tiltak er legitimt
utforme rolleforventninger i omstilling
```

### Praktisk test

```text
Kan vi skrive målet som en regel som gjelder på tvers av saker?
```

Hvis svaret er nei eller usikkert, er målklarheten lavere.

## Separabilitet

Separabilitet handler om om prediksjon kan skilles fra menneskelig vurdering.

### Høy separabilitet

KI-output kan brukes direkte eller nesten direkte, fordi vurderingsdelen er liten, standardisert eller kan håndteres som unntak.

### Lav separabilitet

Menneskelig vurdering er en del av selve beslutningen. Å fjerne mennesket kan svekke ansvar, legitimitet eller kvalitet.

## Tre typer lav separabilitet

### 1. Teknisk separabilitet

KI er ikke presis nok, kan hallusinere, mangler kontekst eller trenger menneskelig kontroll.

### 2. Legitimitetsbasert separabilitet

Et menneske må eie vurderingen, stå ansvarlig og kunne forklare beslutningen.

### 3. Regulatorisk separabilitet

Lov, regelverk eller interne krav krever menneskelig vurdering.

## Operativ regel

I HR, HMS, workforce planning og incident data bør standardantakelsen være:

```text
Lav separabilitet inntil det motsatte er dokumentert.
```
