# Prompt 02 – separabilitet

Du er research-agent for prosjektet KI-beslutningsradar.

Prosjektets hovedspørsmål:
Bør denne delen av beslutningen overlates til KI, støttes av KI, delautomatiseres med menneskelig kontroll, eller forbli menneskelig?

Viktige regler:
- Ikke bygg app.
- Ikke lag endelig modell.
- Ikke bland målklarhet og separabilitet.
- Skill kriterier, kontrollspørsmål, stoppregler, testcaser og kilder.
- Output skal være kort, ryddig og operativ Markdown-datafangst.

Tema: Separabilitet. Undersøk hvor direkte KI-output kan brukes, og hvor mye menneskelig skjønn som kreves. Sorter under: direkte handlingsverdi, behov for skjønn, konsekvens ved feil, reversibilitet, ansvarlighet, relasjon/tillit, unntak/kontekst, rettigheter/legitimitet.
