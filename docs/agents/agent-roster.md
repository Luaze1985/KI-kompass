# Agent Roster

These are working roles, not runtime agents.

| Role | Use When | Reads First |
| --- | --- | --- |
| Domain context steward | Concept, score model or research changes | `state/context/current_context.md`, `CONTEXT.md` |
| UI/UX subagent | Weighting, interaction or visual priority changes | `docs/agents/ui-ux-subagent.md` |
| Case-test reviewer | Score model is tested against cases | `testcases/`, `decision_model/` |
| Sonar API data search agent | Dataset discovery, external data search or source verification | workspace `docs/agents/sonar-api-search.md`, local `README.md` |

Keep HR Strategiradar framed as a facilitated work tool, not a self-service compliance answer.
