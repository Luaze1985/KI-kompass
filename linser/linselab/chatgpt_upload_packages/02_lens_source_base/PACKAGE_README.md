# Package 02: Lens Source Base

Purpose: Provide local primary source material for creating and improving Linselab lenses.

Use this package when:

- normalizing incomplete lenses
- extracting theory, triggers, conflicts, and standard weights
- checking whether a proposed new lens already exists
- preserving local Linselab concepts before importing external method ideas

Priority:

1. `KJERNELINSER.md.txt`
2. `LINSEINDEKS.md.txt`
3. `alle_linser_siste.docx.txt`
4. `LINSEKARTLAG_IKKE_KOMMUNALE_ROLLER.md.txt`
5. `GRUPPE_L_STRUKTURMASTER.md.txt`

Rule: new lenses are allowed only when a case shows a real gap not already covered by these sources.
