# Package 03: External Repo Methods

Purpose: Provide the useful parts of downloaded external repos without making them primary authority.

Included roles:

- AgentForge: method extraction, prompt layering, guardrail checks, evaluator, gap analysis, handoff logic.
- PersonaNexus: advisor profiles, answer formats, boundaries, uncertainty signals.
- Prism: method catalog and shortlist only; content remains license-blocked.

Use this package to improve:

- trigger-to-technique logic
- lens draft quality checks
- case evaluation
- deterministic handoffs
- profile-to-lens tone and boundary mappings

Do not copy Prism library text. Use only the catalog-level status and candidate mappings until license is resolved.
