# KODETABELLER

## Formaal

Denne mappen inneholder avledede YAML- og CSV-tabeller for HR-piloten.
Tabellene brukes til agentstotte, validering og enkel eksport, men de overstyrer aldri Markdown-masterne.

## Styringsregel

- oppdater Markdown-mastere forst
- oppdater deretter modell og kodetabeller
- bruk samme ID-familier som i HR-masterne
- marker lokale forhold som `local_status` i stedet for a bake dem inn som faste sannheter

## Innhold i v1

- `hr_units.*`
- `hr_roles.*`
- `hr_processes.*`
- `hr_skills.*`
- `hr_systems.*`
- `hr_artifacts.*`
- `hr_regulations.*`
- `hr_risks.*`
- `hr_templates.*`
