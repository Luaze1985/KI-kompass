## DEL 3: BRUKSMØNSTRE

### Mønster 1: Hurtigsjekk (2-3 linser)
For enkle saker eller første gjennomlesing.

**Standard kombinasjon:**
- Juristen (hjemmel og risiko)
- En faglinse relevant for tema
- Kritisk venn (svakheter)

---

### Mønster 2: Fullstendig utredning (8-12 linser)
For saker som skal til politisk behandling.

**Anbefalt sekvens:**
1. **Systemteoretiker** → Forstå helheten først
2. **Jurist** → Avklar handlingsrom
3. **Samfunnsøkonom** → Kost-nytte
4. **Etisk filosof** → Verdikonflikter
5. **Maktkartlegger** → Hvem kan stoppe?
6. **Reverseringslinsen** → Kan vi angre?
7. **Oppgaveatomist** → Hva er stegene?
8. **Førstelinjeleder** → Gjennomførbart?
9. **Brukeren** → Hvordan oppleves det?
10. **Kritisk venn** → Stresstest
11. **Stabssjef** → Syntese og anbefaling

---

### Mønster 3: Implementeringsfokus (Gruppe L)
Når beslutningen er tatt og du skal gjennomføre.

**Bruk disse i sekvens:**
1. **Maktkartlegger** → Hvem må med?
2. **Reverseringslinsen** → Pilotmulighet?
3. **Oppgaveatomist** → Mikrooppgaver
4. **Førstelinjeleder** → Kapasitetssjekk
5. **Brukeren** → Brukertest

---

### Mønster 4: Politisk sensitiv sak
Når det er mange interesser og risiko for motstand.

**Bruk disse:**
- Maktkartlegger (hvem kan stoppe?)
- Narrativ-analytikeren (hvilke fortellinger kolliderer?)
- Politisk teoretiker (legitimitet)
- Etiker (prinsipper)
- Kommunikasjonsekspert (budskap)

## DEL 5: POLARITETSKART FOR HOVEDSPENNINGER

### Polaritet 1: Effektivitet ↔ Rettferdighet

```
┌────────────────────────────────────────────────────────────────┐
│                    STØRRE FORMÅL:                              │
│         Bærekraftige tjenester som tjener alle                 │
├────────────────────────────────────────────────────────────────┤
│  EFFEKTIVITET (+)              │  RETTFERDIGHET (+)            │
│  • Mer tjeneste per krone      │  • Alle får lik tilgang       │
│  • Raskere saksbehandling      │  • Svake grupper ivaretas     │
│  • Frigjør ressurser           │  • Tillit til systemet        │
├────────────────────────────────┼───────────────────────────────┤
│  EFFEKTIVITET (-)              │  RETTFERDIGHET (-)            │
│  Overfokus gir:                │  Overfokus gir:               │
│  • Svake grupper faller ut     │  • Ressurssløsing             │
│  • Standardløsninger passer få │  • Lange køer for alle        │
│  • Tap av tillit               │  • Byråkratisk treghet        │
├────────────────────────────────────────────────────────────────┤
│                    DYPERE FRYKT:                               │
│    System som verken leverer eller er rettferdig               │
└────────────────────────────────────────────────────────────────┘

TIDLIG VARSEL (effektivitetsdrift): Klager fra sårbare grupper øker
TIDLIG VARSEL (rettferdighetsdrift): Ventetider/kostnader eskalerer
```
