# ChatGPT Project-startpakke

## Formål

Denne filen beskriver hva som bør lastes inn i ChatGPT Project når sammenslåingsarbeidet skal fortsette der.

## Anbefalt første opplasting

Maks 8-12 filer:

1. `README.md`
2. `AGENTS.md`
3. `00_styring/MANDAT_SAMMENSLAAING.md`
4. `00_styring/KILDEPRIORITET.md`
5. `01_arbeidsflyt/SAMMENSLAAINGSFLYT.md`
6. `01_arbeidsflyt/STOPPUNKTER.md`
7. `03_registre/SOURCE_REGISTER_START.yml`
8. `03_registre/MERGE_BACKLOG.md`
9. `04_case/CASE_TEMPLATE_SAMMENSLAAING.md`
10. `99_styrende_kilder/linser_raad/_RAMMEVERK.md`
11. `99_styrende_kilder/linser_raad/_NY_LINSE.md`
12. `99_styrende_kilder/linser_raad/_BRUK.md`

## Ikke last inn i første runde

- hele repo-inspirasjoner
- alle originaldocx
- arkiv/duplikater
- store kildesamlinger før inventar er laget

## Arbeidsledende prompt

```text
Du er jobbbibliotekar og arkitekt for Linselab.

Bruk startpakken for sammenslåing.
Arbeid kun med registrering, klassifisering og plan for første normalisering.
Ikke lag ferdig bibliotek ennå.
Ikke slett eller overskriv originaler.

Målet er å få eksisterende linser, rådgiverledetekster og styringsfiler inn i ett sporbarhetsbasert system.

Start med:
1. kilderegister
2. klassifisering av instrumenttyper
3. forslag til første 5 linser for normalisering
4. forslag til ett case for test
5. åpne spørsmål før videre arbeid
```
