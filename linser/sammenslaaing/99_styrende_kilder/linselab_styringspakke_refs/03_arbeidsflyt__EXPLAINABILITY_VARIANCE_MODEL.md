# Forklarings-, varians- og gjennomskjæringsmodell

## Formål

Rangere og velge linser ut fra mer enn relevans. En god rådgiverlinse må kunne forklare, utfordre og synliggjøre usikkerhet.

## Hovedscorer

| Kriterium | Spørsmål | Score |
|---|---|---:|
| Forklaringsstyrke | Forklarer linsen hvorfor saken ser ut som den gjør? | 1-5 |
| Teoretisk forankring | Bygger linsen på kjent teori, metode eller fagtradisjon? | 1-5 |
| Empirisk/praktisk støtte | Har linsen støtte i erfaring, praksis eller data? | 1-5 |
| Konteksttreff | Passer linsen til faktisk organisasjon, rolle og situasjon? | 1-5 |
| Handlingsverdi | Fører linsen til bedre valg eller neste handling? | 1-5 |
| Etterprøvbarhet | Kan funn kontrolleres av menneske? | 1-5 |
| Gjennomskjæringsstyrke | Avdekker linsen svake argumenter, gjetting og skjulte premisser? | 1-5 |
| Nyanseevne | Ser linsen viktige mellomposisjoner og unntak? | 1-5 |
| Blindsonedeteksjon | Ser linsen det andre perspektiver overser? | 1-5 |

## Gjennomskjæringsskala

| Score | Betydning |
|---:|---|
| 1 | Gjentar hovedargumentet uten vesentlig kritikk |
| 2 | Finner enkelte svake punkter |
| 3 | Skiller tydelig fakta, antakelser og usikkerhet |
| 4 | Avdekker skjulte premisser, motargumenter og underkommuniserte konsekvenser |
| 5 | Endrer forståelsen av saken ved å vise en oversett mekanisme, risiko eller mulighet |

## Varians mellom linser

| Varians | Tolkning | Handling |
|---|---|---|
| Lav | Linsene peker samme vei | Sjekk om enighet er reell eller om linsene er for like |
| Moderat | Linsene vektlegger ulike hensyn | Bruk syntese og prioritering |
| Høy | Linsene gir motstridende anbefalinger | Løft som beslutningsdilemma |
| Farlig lav | Alle bekrefter samme antakelse uten motstand | Legg inn kritisk venn/gjennomskjæringslinse |
| Produktiv høy | Uenighet viser reell spenning | Dokumenter avveining eksplisitt |
| Støyende høy | Uenighet skyldes uklare premisser | Gå tilbake til fakta og avgrensning |

## Obligatorisk skille i output

Hver linseanalyse skal skille:

```text
Fakta: Dette vet vi.
Antakelse: Dette tror vi, men har ikke dokumentert.
Tolkning: Dette er én mulig forklaring.
Gjetting: Dette er plausibelt, men svakt underbygget.
Gjennomskjæring: Dette er svakt punkt, skjult premiss eller oversett mekanisme.
```
