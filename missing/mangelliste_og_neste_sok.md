# Mangelliste og neste søk

## 1. Mangler som er delvis lukket i denne pakken

| Mangel | Status |
|---|---|
| Separabilitet manglet som fil | supplert med ny datafangst |
| Forklarbarhet + human oversight manglet som fil | supplert med ny datafangst |
| Anti-overreliance manglet som fil | supplert med ny datafangst |
| Samlet syntese manglet | supplert med runde 1-syntese |
| Scoremodell manglet | supplert som foreløpig modell |
| Kildeoversikt manglet | supplert |
| Codex/GitHub-handoff manglet | supplert |

## 2. Mangler som fortsatt må løses

- Skåring og vekting er ikke testet på faktiske caser.
- Beslutningslogg trenger egen datafangst.
- Linse-systemet er ikke operasjonalisert.
- Norsk offentlig sektor trenger flere relevante testcaser.
- HR-caser må kvalitetssikres mot arbeidsrett, diskriminering og personvern.
- Det mangler evalueringssett med «riktige» og «feil» KI-roller.

## 3. Tema som bør undersøkes videre

1. Datatilsynet og EDPB om automatiserte individuelle avgjørelser.
2. Digdir/KI Norge om ansvarlig KI i offentlig sektor.
3. NIST AI RMF MAP/MEASURE/MANAGE på detaljerte underpunkter.
4. ISO/IEC 23894 om AI-risikostyring.
5. Forskning på appropriate reliance og automation bias.
6. Beslutningslogg, audit trails og dokumentasjonskrav.
7. Human oversight i praksis: kompetanse, tid, myndighet og eskalering.

## 4. Dette bør ikke forsinke zip

- Endelig vekting.
- Appdesign.
- Grafikk/dashboard.
- Full litteraturgjennomgang.
- GitHub-repo.

## 5. Anbefaling

Klar for zip som **runde 1-arbeidspakke**, ikke som ferdig beslutningsmodell.
